# ZonGrabber - 亚马逊商品采集助手 v1.4.0

一个专业的Chrome扩展程序，支持亚马逊商品详情页和列表页的智能数据采集。

## 🚀 核心功能

### 📊 双模式采集
- **单品采集**: 商品详情页深度数据采集
- **列表采集**: 搜索结果页批量商品采集
- **智能切换**: 自动识别页面类型，无需手动切换

### 🎯 智能过滤系统
- **销量过滤**: 基于评论数量设置最低销量阈值
- **评分过滤**: 过滤低评分商品，确保质量
- **评论数过滤**: 确保商品有足够的用户反馈
- **品牌筛选**: 支持指定品牌名称精确筛选
- **智能排序**: 按销量、评分、价格多维度排序

### 📋 列表页面支持
- **搜索结果页**: `/s?` 搜索结果批量采集
- **分类页面**: `/b/` 分类商品批量采集
- **品牌店铺**: `/stores/` 品牌店铺商品采集
- **多重选择器**: 适应亚马逊反爬机制和页面变化

### 💾 数据管理优化
- **清洁URL**: 自动清理商品链接，只保留核心部分
- **联盟标识**: 自动添加设置的联盟标识
- **简化导出**: 直接导出商品数组，无冗余信息
- **实时预览**: 侧边栏实时显示采集结果

### 🔄 用户体验
- **自动检测**: 页面切换时自动更新功能按钮
- **响应式设计**: 适配各种屏幕尺寸
- **状态指示**: 清晰的采集状态和进度提示
- **错误处理**: 智能错误处理和用户友好提示

## 📦 安装方法

1. 打开Chrome浏览器
2. 访问 `chrome://extensions/`
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择ZonGrabber文件夹
6. 插件安装完成

## 🔧 使用方法

### 📊 单品采集模式
1. **打开商品详情页** - 访问任意亚马逊商品详情页面（包含 `/dp/` 的URL）
2. **启动插件** - 点击浏览器工具栏中的ZonGrabber图标
3. **自动识别** - 插件自动识别为单品模式，显示"📊 采集单品"按钮
4. **采集数据** - 点击"采集单品"按钮获取详细商品信息
5. **查看结果** - 在不同标签页查看详细信息
6. **导出数据** - 点击"📥 导出单品"保存数据

### 📋 列表采集模式
1. **打开列表页面** - 访问亚马逊搜索结果页、分类页或品牌店铺
2. **启动插件** - 点击浏览器工具栏中的ZonGrabber图标
3. **自动识别** - 插件自动识别为列表模式，显示"📋 采集列表"按钮
4. **设置过滤条件**:
   - 最低销量（评论数）: 例如设置为 10
   - 最低评分: 例如设置为 3.5
   - 最低评论数: 例如设置为 5
   - 品牌筛选: 例如输入 "Contigo"
   - 排序方式: 选择按销量、评分或价格排序
5. **批量采集** - 点击"采集列表"按钮获取符合条件的商品
6. **导出数据** - 点击"📋 导出列表"保存筛选后的商品数据

### 🔄 智能切换
- 在不同亚马逊页面间切换时，插件会自动检测页面类型
- 按钮文字和功能会自动更新，无需手动刷新
- 支持的页面类型：
  - 商品详情页: `amazon.com/dp/ASIN`
  - 搜索结果页: `amazon.com/s?k=关键词`
  - 分类页面: `amazon.com/b/分类ID`
  - 品牌店铺: `amazon.com/stores/品牌`

## 📊 采集的数据字段

### 基础信息
- ASIN、商品标题、品牌
- 当前价格、原价、价格区间
- 评分、评论数量、销量排名

### 变体信息
- 颜色选项、尺寸选项、样式选项
- 图案选项、材质选项、其他选项

### 联盟信息
- SiteStripe状态、商品分类
- 佣金率、计算的销售佣金

### 评论数据
- 评论内容、评分、作者
- 发布日期、验证状态、有用投票

### 商品详情
- 商品特性、技术规格
- 商品描述、分类信息
- 库存状态、配送信息

## 🛠️ 故障排除

**Q: 插件无法加载？**
A: 确保logo.png文件存在，检查manifest.json语法

**Q: 数据采集失败？**
A: 确认在亚马逊商品页面，等待页面完全加载

**Q: 图标不显示？**
A: 重新加载插件，确认插件已启用

**Q: 采集超时？**
A: 刷新页面后重试，检查网络连接
